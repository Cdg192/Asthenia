## Rescue Ptouneigh from the Evil Sorceress!

**Play the game here:** [🎮 Click to Play ASTHENIA](https://htmlpreview.github.io/?https://github.com/Cdg192/throughthetreess/blob/main/index.html)

### About the Game
Join Caleb and his sidekick Chazz on an epic quest to rescue Ptouneigh from the clutches of the evil sorceress Tracey. Armed with a Gibson Guitar and unwavering determination, battle through cultists and face off against the Sionist conspiracy.

### Features
- 🎨 3D graphics with Three.js
- 📐 Top-down isometric view
- ⚔️ Turn-based combat system
- 🎬 Combat animations
- 🎵 Gibson Guitar-powered attacks
- 📊 Real-time battle log
